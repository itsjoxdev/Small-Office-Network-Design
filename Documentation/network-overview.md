# Small Office Network Overview

## Overview

This project demonstrates the design and implementation of a small office network using Cisco Packet Tracer.

The network is designed for a company with six employees and provides reliable wired and wireless connectivity, shared resources, and centralized network services.

---

## Business Requirements

The network should provide:

- Internet access for all employees
- Wired network connectivity
- Wireless network access
- Shared network printer
- Centralized server
- Automatic IP address assignment (DHCP)
- Static IP addresses for infrastructure devices
- Secure and scalable network design

## Network Topology

The network follows a **Star Topology**, where all devices are connected to a central Cisco Switch.

Benefits include:

- Easy troubleshooting
- High reliability
- Scalability
- Better performance
- Simplified management
